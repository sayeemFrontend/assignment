# I have initialized React App using vite

# Installation

1. npm create vite@latest assignment
2. cd assignment
3. npm install
4. npm run dev

# Run Application

1. git clone https://github.com/sayeemFrontend/assignment.git
   (make sure current brach is master)
2. cd assignment
3. npm install
4. npm run dev

# Production

1. npm run build
2. serve dist
